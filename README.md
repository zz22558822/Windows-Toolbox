## 使用介紹:
本程序採用 PyQt 6 建置 GUI，功能如下:

1. 電源休眠一鍵設定 (AC模式)
2. 簡易獲取 Windows 各式授權資訊
3. 檔案總管隱藏資料夾 (下方有分頁可選隱藏、顯示)
4. 解鎖舊版 相片顯示器
5. 若系統異常可運行 SFC修復
6. Windows Home 安裝 本機群組原則(gpedit.msc)
7. Netplwiz 顯示自動帶入密碼選項
